## Diagramas de Estados — Sistema IURIS

**DE01 — Estado del Caso Legal**
Activo → En Audiencia → En Espera → Cerrado → Archivado

**DE02 — Estado de una Audiencia**
Programada → En Curso → Concluida
(alternativas: Suspendida, Cancelada)

**DE03 — Estado de un Documento**
Borrador → Publicado → Firmado → Archivado
(alternativa: Eliminado)

**DE04 — Estado de una Cita**
Programada → Confirmada → En Curso → Completada
(alternativas: Reprogramada, Cancelada)

**DE05 — Estado de un Mensaje**
Enviado → Entregado → Leído

**DE06 — Estado de un Reporte**
Solicitado → Generado → Exportado
