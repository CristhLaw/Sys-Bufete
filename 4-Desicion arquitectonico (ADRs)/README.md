# Decisiones Arquitectónicas (ADRs)

Esta carpeta documenta las decisiones arquitectónicas relevantes del
proyecto IURIS, usando el formato ADR (Architecture Decision Record).

## Convención de nombres

`ADR-NNN-titulo-corto-en-minusculas.md`, numerados secuencialmente en
el orden en que se toman las decisiones.

## Estructura de cada ADR

1. **Título** — decisión resumida en una línea.
2. **Estado** — Propuesto / Aceptado / Rechazado / Reemplazado por ADR-XXX.
3. **Contexto** — la situación que obliga a decidir.
4. **Decisión** — qué se decidió hacer.
5. **Alternativas consideradas** — qué otras opciones se evaluaron y por qué se descartaron.
6. **Consecuencias** — impactos positivos y negativos aceptados.
7. **Referencias** — documentos o diagramas relacionados.
